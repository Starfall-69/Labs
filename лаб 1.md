# Лабораторная работа №1
## Изучение базовых возможностей MS Visual Studio
### Цель: изучить основные возможности создания и отладки программ в IDE MS Visual Studio.
### Ход работы:
1. Создание консольного приложения С++. 
 + Чтобы создать консольное приложение в MS Visual Studio нужно открыть программу MS Visual Studio. Когда высветится окно, нужно нажать "Создание проекта".(скрин1) 
 + В представленом меню выбираем пункт "Консольное приложение" и нажимаем "Далее"(скр2)
 + В следующем окне в 1м шаге даем имя проекту, во 2м шаге мы можем указать путь к проекту, в 3м шаге мы можем дать имя решению, после проделанного нажимаем "Создать".( скр3)
 + Перед нами открывется окно для редактирования и создания кода, языка программирования С++. (скр4) 
 2. Изменение цветовой схемы (оформления) среды.
 + В окне редактирования и создания кода выбираем кнопку "Средства". Перед нами появиться выпадающий список и в нем выбираем пункт " Параметры" (скр5)
 + Откроется окно "параметры" в нем сразу будет открыт пункт "общие" 
 В нем ищем пункт "цветовая тема" и вбирам ту, которая больше вам нужна и нажимает "ок" (скр 6) 
 + Результат выбора темной темы. (скр 7) 
 3. Закомментирование/раскомментирование блока кода средствами VS.
 + Чтобы закомментировать блок кода нужно его выделить и использовать комбинацию `Ctrl`+`K`,`Ctrl` + `C`. (скр 8)
 + Чтобы раскомментировать мы должны выделить закомментированный блок кода и использовать комбинацию `Ctrl` + `K`, `Ctrl`+ `U`. 
 4. Как открыть в проводнике Windows папку с проектом средствами VS.
 + В VS нажимаем на кнопку "Вид" перед нами появляется выпадающий список, в нем выбираем "Обозреватель решений". После этого появляется окно "обозреватель решений". (скр 9)
 + Нажимаем правой клавишей мыши на свой проект и в контекстном меню выбираем пункт "Открыть папку в проводнике".(скр 10) 
 5. Какое расширение файла-проекта используется в VS.
 + Используется расширение SLN (скр 11).
6. Запуск кода без отладки.
 + Первый способ: находим кнопку сверху "отладка" и нажимаем "Запуск без отладки". (скр 12)
 + Второй способ: использовать комбинацию `Ctrl` +`F5`.
 7. Запуск кода в режиме отладки.
 + Нажимаем кнопку "Отладка" и выбираем пункт "Начать отладку". (скр 13) 
 + Нажимаем кнопку `F5`.
 8. Устанавливаем/ убираем точку останова. 
 + Чтоб установить точку останова нажимаем правой клавишей мыши на той строке, на которой мы хотим поставить точку останова и в контекстном меню выбираем пункт точки останова.Далее выбираем пункт "Поставить точку останова". (сскр 14)
 + Чтобы убрать точку останова, нажимаем по ней правой клавишей мыши и выбираем пункт "Удалить точку останова".
 9. Создадим программу со следующим кодом и поставим точки останова на строках 5, 6. Выставим конфигурацию сборки Debug запустим отладку  (скр 15)
  + значение i в 5й строке = -858993460;
  + значение i в 6й строке = 5;(скр 16)
10. Создадим программу со следующим кодом и поставим точки останова на строках 5, 6. Выставим конфигурацию сборки Debug запустим отладку  (скр 17)
  + значение i в 5й строке = -9.2559631349317831e+61;
  + значение i в 6й строке = 5.0000000000000000 ;(скр 18).
